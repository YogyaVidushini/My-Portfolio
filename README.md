# My-Portfolio
A responsive personal portfolio website designed to showcase my web development projects, skills, and experience.



## Overview

This portfolio presents my work in a professional and organized manner. It focuses on responsive design principles, usability, and clean UI layout to ensure compatibility across all devices.


## Features

- Fully responsive design for all screen sizes  
- Modern hero section with background image  
- Project showcase using card-based layout  
- Skill visualization using progress bars  
- Fixed navigation bar with smooth scrolling  
- Clean and minimal user interface  
- Contact form layout for communication
